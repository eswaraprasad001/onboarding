## 📋 Quick Start Options

### Option 1: One-Click Setup (Windows)
Double-click `quick-start.bat` in the onboarding-app folder. This will:
- Check Node.js installation
- Install dependencies automatically
- Start the development server
- Open the application in your browser

### Option 2: One-Click Setup (Mac/Linux)
Run `./quick-start.sh` in the terminal from the onboarding-app folder:
```bash
cd onboarding-app
./quick-start.sh
```

### Option 3: Manual Setup
```bash
cd onboarding-app
npm install
npm run dev
```

## 🌐 Application Access

Once started, the application will be available at:
- **Primary URL**: http://localhost:3000
- **Alternative URLs**: http://localhost:3001, http://localhost:3002 (if port 3000 is busy)

The application automatically finds the next available port if the default is in use.
